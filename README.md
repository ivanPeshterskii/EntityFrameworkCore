# 📚 Entity Framework Core – June 2026
### SoftUni C# Path – Database Fundamentals

Welcome to my GitHub repository for the **Entity Framework Core** course from the **C# Full-Stack Developer** learning path at SoftUni.

This repository contains my solutions to all **labs**, **exercises**, **workshops**, **exam preparation tasks**, and **exam problems** throughout the course.

---

## 📖 About the Course

The **Entity Framework Core** course provides practical knowledge for working with databases in C# applications using modern ORM technologies. During the course, I study:

- Object-Relational Mapping fundamentals
- Working with Entity Framework Core
- Database-first and code-first approaches
- Entity relations and navigation properties
- LINQ queries
- Advanced querying techniques
- Migrations in Entity Framework Core
- JSON processing
- XML processing
- Using Entity Framework Core in ASP.NET applications
- Best practices and application architecture
- Working with NoSQL concepts
- Exam preparation and real-world database tasks
